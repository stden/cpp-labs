Программа демонстрирует работу мультиметодов на примере геометрических фигур.

Использование:

1) нарисовать точки и отрезки;
2) выбрать инструмент "Выделение" и выделить некоторое количество точек, отрезков;
3) запустить мультиметод через контекстное меню ( правая кнопка мыши );
4) радоваться.

В зависимости от количества параметров и их типов, вызывается одна из функций
(см. LRESULT CMainFrame::OnMultimethod(WPARAM, LPARAM)):

         -рисующая отрезок из 2 точек.
         -рисующая треугольник из 3 точек;
         -рисующая треугольник из точки и отрезка;
         -рисующая прямоугольник из 2 отрезков.
